# Recursos Humanos - NexusFlow Technologies

Documentación ficticia del dominio de Recursos Humanos para el corpus RAG.

## Documentos

- `09_Manual_Bienvenida_NexusFlow.pdf`: incorporación y primeros pasos.
- `10_Politica_Trabajo_Remoto_NexusFlow.docx`: trabajo remoto, coordinación y seguridad.
- `11_Matriz_Cargos_Competencias_NexusFlow.xlsx`: familias de cargos y catálogo de competencias.
- `12_FAQ_Recursos_Humanos_NexusFlow.csv`: preguntas frecuentes normalizadas.
- `13_Codigo_Conducta_NexusFlow.md`: conducta profesional, información y reportes.
- `14_Corpus_RH_NexusFlow.json`: chunks normalizados para recuperación.

## Convenciones RAG

- Prefijos: `RH-ONB-*`, `RH-REM-*`, `RH-FAQ-*`, `RH-COND-*`, `ROLE-*`, `COMP-*`.
- Metadatos: `department=rh`, `locale=es-CL`, `version=1.1`.
- La matriz de cargos describe roles ficticios; no sirve para evaluar personas.
- Los contratos y políticas oficialmente vigentes prevalecen sobre este material demostrativo.
